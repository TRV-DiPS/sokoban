# 关于自定义关卡

在JSON最后加上自己设计的关卡即可（请添加至最新的json中）

### 例子

原始json

```
[
  {
    "level": "1",
    "author": "by 月银秋(QQ2859040107)",
    "map": [
      [2, 2, 2, 2, 2],
      [2, 5, 3, 4, 2],
      [2, 1, 2, 2, 2],
      [2, 2, 2, 0, 0]
    ]
  }
]

```

在第一关的基础上添加第二关：

你添加的数据应该像这样
```
,
  {
    "level": "2",
    "author": "你的信息",
    "map": [
      [2, 2, 2, 2, 2],
      [2, 5, 3, 4, 2],
      [2, 1, 2, 2, 2],
      [2, 2, 2, 0, 0]
    ]
  }
```

最后结果

```
[
  {
    "level": "1",
    "author": "by 月银秋(QQ2859040107)",
    "map": [
      [2, 2, 2, 2, 2],
      [2, 5, 3, 4, 2],
      [2, 1, 2, 2, 2],
      [2, 2, 2, 0, 0]
    ]
  },
  {
    "level": "2",
    "author": "你的信息",
    "map": [
      [2, 2, 2, 2, 2],
      [2, 5, 3, 4, 2],
      [2, 1, 2, 2, 2],
      [2, 2, 2, 0, 0]
    ]
  }
]

```
