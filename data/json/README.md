# 关于自定义关卡

在JSON最后加上自己设计的关卡即可（请添加至最新的json中）

不会自己提交的请加群提交，群号：574298353

地图数组的数字含义

 - `0` 空地
  
 - `1` 人物位置 
  
 - `2` 墙壁
  
 - `3` 箱子
  
 - `4` 终点
 
 - `5` 道路
 
 - `6` 箱子已在终点

### 例子

原始json

```
[
  {
    "level": "1",
    "author": "by 月银秋(QQ2859040107)",

]

```
